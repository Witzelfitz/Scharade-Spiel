4. Google Fonts API

Diese Website verwendet Schriftarten von Google Fonts. Beim Aufruf einer Seite lädt Ihr Browser die benötigten Webfonts in den Browser-Cache, um Texte korrekt anzuzeigen.

Dabei wird eine Verbindung zu den Servern von Google aufgebaut. Google kann dabei Ihre IP-Adresse erfassen. Die Nutzung erfolgt zur ansprechenden Darstellung der Inhalte.

Weitere Informationen zu Google Fonts finden Sie unter: developers.google.com/fonts/faq

Die Datenschutzerklärung von Google finden Sie hier: policies.google.com/privacy
